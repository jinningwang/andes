=============
Input formats
=============

ANDES currently supports the following input formats:

- ``.xlsx``: Excel spreadsheet file with ANDES data
- ``.json``: JSON plain-text file with ANDES semantics
- ``.raw`` and ``.dyr``: PSS/E raw and dyr files
- ``.m``: MATPOWER case file
